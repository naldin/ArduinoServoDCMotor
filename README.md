# ArduinoServoDCMotor
An Arduino servo motor with DC motor and potentiometer.

## Servo_DC_Motor_Pot
Using only proportionality for read/target value.

## Servo_DC_Motor_Pot_Prop
Using proportionality for read/target value and pwm.
